## 🚀 DÉPLOIEMENT GITHUB PAGES (5 ÉTAPES)

### 1️⃣ Créer le repository
```
1. Allez sur github.com
2. Créez repo: "username.github.io"
3. Public + Create
```

### 2️⃣ Uploader les fichiers
```
Téléversez tous les fichiers dans la racine
```

### 3️⃣ Vérifier l'accès
```
https://username.github.io
```

### 4️⃣ Configurer Stripe
```
1. Allez sur stripe.com → Dashboard
2. Créez Payment Link
3. Admin > Paramètres > Collez l'URL
```

### 5️⃣ Configurer Formspree
```
1. Allez sur formspree.io
2. Créez formulaire → Copiez l'ID
3. Admin > Paramètres > Collez l'ID
```
